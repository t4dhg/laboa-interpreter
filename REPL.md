# REPL Erabilera Gida

Laboa REPL (Read-Eval-Print Loop) abiarazteko, exekutatu:

```bash
node laboa.js repl
```

Ondoren, Laboa kodea modu interaktiboan idatzi dezakezu:

```
laboa> ageri('Kaixo mundua!')
Kaixo mundua!
laboa> izan x = 5
laboa> izan y = 10
laboa> ageri(x + y)
15
laboa> irten
```

`exit` erabili REPLetik irteteko.
